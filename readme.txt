Window 安裝說明
1 執行dist/mongoose.exe
2 打開Chrome ,在網址輸入 http://localhost:8000
3 按f11啟動全螢幕


Mac 安裝說明
1 在dist資料夾上按右鍵,選擇 "新增位於檔案夾位置的終端機視窗"
2 在終端機輸入(或貼上)以下文字後按enter:  

ruby -run -e httpd . -p 8000

3 打開Chrome ,在網址輸入 http://localhost:8000
4 按Mac：按下⌘ + Ctrl + F 鍵 啟動全螢幕

*如果mac版本過舊，執行第一步後，需先輸入:

xcode-select --install

安裝完後，在執行第2步操作即可。
